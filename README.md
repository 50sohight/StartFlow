## Правила внесения изменений в кодовую базу:  
1. Создаем новую ветку:  
   ```bash  
   git checkout --orphan new_feature  
   ```  
2. Делаем локальный коммит:  
   ```bash  
   git add .  
   git commit -m "feature"  
   ```  
3. Создаем PR  
   ```bash  
   git push origin new_feature -f # f означает feature  
   ```  
P.S это шутка ребята не делайте так
